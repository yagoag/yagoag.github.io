HouSplit

Ver no GitHub

HouSplit é um sistema de gerenciamento de contas baseado em PHP e MySQL para pessoas que dividem casa e contas, especialmente estudantes universitários. Originalmente, ele foi criado para o uso por Repúblicas, mas ele também pode ser utilizado em muitas outras situações nas quais as pessoas dividem contas e/ou emprestam dinheiro umas para as outras. Alguns outros lugares nos quais o sistema pode ser utilizado incluem residências/alojamentos universitários e casas de fraternidade (como as norte-americanas).

A ideia para este sistema se iniciou quando a minha República precisou uma forma de dividir contas e controlar pagamentos de forma simples e fácil, por termos problemas ao atualizar os saldos de cada um manualmente na tabela que utilizávamos devido a erros de cálculo. O sistema é baseado em transações de três tipos principais: empréstimos, pagamentos e contas.

Pluto Organizador de Projetos
Ver no GitHub

Pluto é um projeto que criei com o meu amigo Leonardo (Leonardobl) como um projeto para uma de nossas disciplinas. No momento em que o professor disse que o projeto seria livre, começamos a pensar em possíveis projetos, e após certo tempo resolvemos iniciar o desenvolvimento deste. A ideia do projeto veio pois o Leonardo estava trabalhando com telas Kanban, utilizadas no modelo de desenvolvimento ágil e pensou que seria muito interessante poder criar telas Kanban virtuais para cada um dos seus projetos. No Pluto, através de um nome de usuário e senha, você pode acessar a sua conta com os grupos de projetos dos quais você faz parte. Dentro dos grupos, você pode acessar a tela Kanban de cada um dos projetos.

O projeto foi desenvolvido utilizando PHP junto ao onipresente HTML & CSS, sem o uso de nenhum tipo de framework.

Calculadora IRA

Ver no GitHub

Calculadora IRA é um website - atualmente desativado - para calcular o Índice de Rendimento Acadêmico (mais conhecido pela sigla IRA) em minha universidade.

O projeto inicial foi um website em HTML, CSS e JavaScript utilizando jQuery. Muitos estudantes ficaram interessados em um aplicativo para poder gerenciar este cálculo em seus celulares, porém com a proposta da mudança para o Índice de Progressão Acadêmica (IPA), deixei esse projeto de lado momentaneamente.

Website do LIA

Este projeto é o protótipo de um projeto de Iniciação Científica desenvolvido no Laboratório de Interação Avançada (LIA) em minha universidade, junto com o George (brucemacho). O objetivo do projeto era modelar e implementar um aplicativo para tablets com Android para simular uma prancheta de táticas, muito utilizada por técnicos de basquete. A ênfase do design do aplicativo foi no uso do Terceiro Paradigma da interação humano-computador.

O protótipo foi implementado em duas versões - uma simples e uma completa, - ambas desenvolvidas em Java.
